## About The Project

This is a project for the Web Data Management Course CSE 5335 at UTA under Dr. Elizabeth Diaz. 
Tools Used
- HTML 5
- CSS3
- Vanilla JS
- Laravel 
- MySql
- Hosted on localhost

You will need to have installed Laravel beforehand -- please go to https://laravel.com/docs/7.x/installation for more instructions
WDMProj is the zip file which is the developed laravel project.
Use this to deploy the project.
